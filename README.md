# Nectar - The best python GUI Framework

<img width="1198" height="719" alt="Screenshot 2026-03-07 213730" src="https://github.com/user-attachments/assets/c387bdb1-262f-4309-a5ed-7e1153a8dd78" />

NectarUI is a GUI Framework for building Python Apps similar to Fluent apps or ides

# Example

```
from NectarUI.InitWindow import NectarWindow

window = NectarWindow(
    use_custom_titlebar=True,
    titlebar_bg="#808080",
    titlebar_height=50
)
window.start()
```
